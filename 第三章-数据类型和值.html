第三章-数据类型和值

  • 三种基本数据类型-数字（number）、文本（string）、布尔值（boolean）
  • 两种小数据类型-空（null）、未定义（undefined）
  • 复合数据类型：对象（object）
    • 已命名的值的无序集合
    • 有编号的值的有序集合=数组
    • 具有可执行代码的对象=函数-特殊对象
    • 其他专用对象：新的对象类（class）

      • 3.1数字（number）

      • 和其他程序设计语言不同，并不区分整数型和浮点型数值
      • JavaScript中所有数字都是浮点型表示
      • 采用IEEE754标准定义的64位浮点格式，有最大值和最小值
      • 数值直接量的形式：
        • 整型直接量
        • 八进制和十六进制的直接量
        • 浮点型直接量
        • 使用：
          • 基本算术运算：直接通过+-*%
          • 大量算术运算：使用Math对象属性
          • toSting()可将数字（2-36）转为二进制表示；直接调用数值使用该法，必须加括号；
        • 特殊的数值：
          • 超出JavaScript表示的最大值表示为Infinity；反之为-Infinity
          • 0/0会产生未定义的结果输出为NaN；需要isNaN()来检测；isFinite()检测一个数字是否为NaNcy、+无穷、-无穷

      3.2字符串

      • 由单引号或双引号包括的unicode字符序列
      • 字符串直接量必须写在一行，换行会被截断，必须换行需要使用\n
      • 引号必须使用英文形式
      • 字符串中含有英文反引号，需要加\隔离，否则被隔断
      • 同时使用JavaScript和HTML时，最好对JavaScript采用一种引用方式，对HTML使用另一种引用方式
      • 字符串直接量的转义序列
        • \表示在字符串中无法出现的字符
        • \n 换行-（\u000A）......等13个
        • \#=#
      • 字符串的使用
        • 以+拼接
        • 以变量 s.length获取长度
        • 以变量 s.charAt(s.length-1)获取最后一个字符
        • 以s.substring(1,4)获取第234个字符
        • 以s.indexOf('a')查询a的位置

      3.3 布尔值

      • 只有true和false两个
      • true等价1，false等价0

      3.4 函数

      • 可执行的代码段，一次定义多次调用
      • javascript中函数是一个真正的数据类型
      • 可被存储在变量、数组、对象中
      • 函数直接量
        • 函数直接量是用关键字function后加可选函数名、括号参数列表、花括号函数体定义的
        • 函数定义法 function square(x){return x*x}
        • 函数直接量定义(拉姆达函数) var square = function(x){return x*x}【重要】
        • 拙劣的定义法 var square = new Function("x","return x*x;");通常无用

      3.5 对象（object）

      • 已命名的数据集合
      • 已命名的数据作为对象属性（对象的域）
      • 要引用一个对象的属性就必须引用对象，在后加点和属性名，例如img.width
      • 属性可以是任何类型的数据包括数组、函数、以及其他对象
      • js对象可作为关联数组使用，能将任意的数据值和任意的字符串关联起来，访问使用img["width"]【重要】
      • 创建对象
        • var o = new Object();
        • var now = new Date();
        • var parrten = new RegExp("\\sjava\\s","i");
      • 对象直接量
        • 属性值不必是常量，可以是任意的javascript表达式

      3.6 数组（array）

      • 和对象一样是数值的集合
      • 和对象不同的是对象每个数值都有一个名字，数组每个数值都有一个数字（下标）
      • 获取数组的值，使用arr[i],i从0开始，非负
      • 数组可以存放任何一种javascript数据类型，包括对其他数组对象或函数的引用
      • 常规数组以数字为下标，关联数组以字符串为下标
      • 数组的创建
        • 使用构造函数Array()来创建数组；var a =new Array()
        • 或者创建、初始化一体形式，var a = new Array(1.02,"js",true,{x:1,y:2});
        • 创建10个未定义的新数组，var ar = new Array(10);
      • 数组直接量
        • 创建和初始化可以直接实现，var a = [1.2,"js",true,{x:1,y:2}]
        • 数组直接量也可以嵌套 var c = [[1,2,3],[2,4,6]];
        • 数组直接量元素不限于常量，可以是任意的表达式
        • 数组直接量可以存放未定义的元素，使用，省去即可，var aa=[1,,,5]

      3.7 null

      • 是对象类型的一个特殊的值，表示无值，无对象
      • 有别于其他所有的值
      • 如果一个变量的值是null，即表示它的值不是有效的对象、数组、数字、字符串、布尔值

      3.8 undefined

      • 使用了一个并未声明的变量时，
      • 使用了已经声明但尚未赋值的变量时，
      • 使用一个并不存在的对象属性时
      • undefined和null值不同，但==将两者看做相等，例如nt.pro==null
      • 区分undefined和null，可以使用===或typeof
      • 无法确认实现有变量undefined，可以声明但不初始化即可 var undefined;

      3.9 Date对象

      • 日期和时间并不属于以上的基本类型
      • 可以使用new 和构造函数Date()，来创建date对象，
      • Date对象还定义了函数，可将日期转为毫秒

      3.10 正则表达式

      • 用于匹配、查找、替换
      • 用RegExp对象表示，可使用RegExp()构造函数来创建
      • 和Date一样，不属于基本数据类型，但所有javascript实现都支持的特殊对象类型【重要】
      • 与Date不同，RegExp有直接量语法
      • 表达式语法复杂

      3.11 Error对象

      • 发生错误时，解释器会抛出某个类的对象，每个Error对象含有一个message属性，表示该错误的信息
      • 使用throw、try语句
      • 预定义错误对象类型有Error、EvalError、RangeError、ReferenceError、SyntaxError、TypeError、URIError

      3.12 基本数据类型的包装对象

      • javascript可以灵活的将一种类型的值转换为另一种类型
      • 在对象环境中使用字符串时，javascript会为字符串内部创建一个string包装对象
      • 访问一个字符串长度时，s.length；String对象瞬时存在，即为瞬态对象转换
      • 如果需要显式使用String，必须创建非瞬态对象，使用var S = new String('hello');
      • 使用String对象时，用+操作符，会有一个瞬态的基本字符串被创建，便于执行字符串拼接
