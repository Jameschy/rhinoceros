9.1 数组和数组元素

  • 数组的创建 
                    
            //无参数的调用

9.2 数组的方法

  • join()方法 
                    
                    var a = [1,2,3];
                    //将数组所有元素转成字符串，然后连接起来
                    var s = a.join(); //s="1,2,3";
                    s = a.join(',');  //s="1, 2, 3"
  • reverse()方法 
                    
                    var a = new Array(1,2,3);
                    //将数组所有元素转成字符串，然后连接起来
                    var s = a.reverse().join();  //s="3,2,1"
  • sort()方法： 
                    
                    var a = new Array(1,2,3);
                    //将数组所有元素转成字符串，然后连接起来
                    var s = a.reverse().join();  //s="3,2,1"
