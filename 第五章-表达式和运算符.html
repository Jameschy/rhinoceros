第五章-表达式和运算符

5.1 表达式（expression）

  • 直接量表达式就是直接量本身；变量表达式的值则是该变量所存放或引用的值。 
                直接量表达式和变量表达式
                
                   1.7                  //数字直接量
                    "java"              //字符串直接量
                    true                //布尔值直接量
                    null                //null直接量
                    /java/              //正则表达式直接量
                    {x:2，y：2}         //对象直接量
                    [1,2,3,4,5]         //数组直接量
                    function (){}       //函数直接量
                    i                   //变量i
                    sum                 //变量sum
  • 复合表达式 
                多个表达式组合
                

                    i + 1.7              //
                    (i + 1.7) - sum      //字符串直接量

5.2 运算符（）

  • 大部分是符号表示，少数是关键字，如delete、instanceof
  • 大部分是二元运算符，符号将两个表达式合并为一个复杂表达式
  • 一元运算符，取反（-3），取非（！true）
  • 三元运算符（ternary operator），a条件？b条件：结果
  • 有的运算符会进行类型转换
    • 74page
  • 运算符优先级（precedence）； 
                运算符优先级-加、乘
                
                    w = x+y*z；//先乘后加
                    w = (x+y)*z；//先加后乘
  • 运算符的结合性
    • 二元运算符从左到右
    • 一元、三元、赋值运算符从右到左

5.3 算数运算符

  • 加法运算符，对数字进行算数加法运算，对字符串进行拼接，如果其中一个为对象，则会将对象转为数字或字符串，通过valueof或toString 执行
  • 减法、乘法、除法运算符，非数字会被转为数字处理
  • 模运算符%，%

5.4 相等运算符

  • "="赋值，"=="等于，"==="完全等同
  • 避免版本问题，script标签内，language不可使用javascript1.2
  • !=（不等运算符，两个值相等返回false），！==（不等同运算符，两个值完全相等返回false），

5.5 关系运算符

  • <,>,<=,>=
  • 字符串比较时，所有大写字母小于所有小写字母
  • in运算符，要求左边的元是一个字符串或可以被转为字符串，右边的元是对象或数组
  •                 in运算符
                
                    var point = {x:1,y:1};          //定义个对象
                    var has_x_coord = 'x' in point; //true
                    var has_y_coord = 'y' in point; //true
                    var has_z_coord = 'z' in point; //false，不是point的维点
                
        
  •                 instandceof运算符，要求左边的元是一个对象，右边的元是对象类的名，左边对象是右边类的实例，则为false；如果右边元不是对象，返回错误
                
                    var d = new Date();          //定义个对象
                    d instandceof Date;          //true
                    d instandceof Object         //true
                    d instandceof Number         //false
                    var a = [1,2,3]              //数组直接量法创建数组
                    a instanceof Array           //true a是一个数组
                    a instanceof Object           //true 数组也是个对象
                    a instanceof RegExp           //false 数组不是正则

5.6 字符串运算符

  • 数字字符串与数字+运算时，会将数字转为字符串再拼接，等于两个字符串拼接运算
  • 字母比较，使用的是Unicode字符编码
  • +运算在字符串和数字综合运算时；结果依赖操作执行的顺序 
                
                 var s = 1+2+'hello'; //"3 hello";
                 var s = 'hello' + 1 + 2; //"hello 12";

5.7 逻辑运算符

  • &&，与AND；两元为true即为true；
  • ||，或OR；两元一者为true即为true
  • 使用&&和||时，右边运算应避免产生副作用（赋值、递增、递减、函数调用）
  • ！，非，对布尔值取反
