8.2构造函数

8.2

  • 要创建已经定义过的带有属性（如width、height）的对象，需要编造一个构造函数，在新的对象中创建并初始化这些属性。构造函数是具有两个特性的JavaScript函数：
    • 由new运算符调用
    • 传递给他的是一个对新创建的空对象的引用，将该引用作为关键字this的值，而且他还要对新创建的对象进行适当的初始化
                    
                    //Rectangle对象的构造函数
                    fuction Rectangle(w,h){
                        this.width = w;
                        this.height = h;
                    }
                    var rect1 = new Rectangle(2,4);
                    var rect2 = new Rectangle(8.5,11);

    构造函数通常没有返回值，他们只是初始化由This值传递进来的对象

8.3 方法

  • 通过对象调用的JavaScript函数，即为方法。
    如果有一个函数f和对象o，可以如此定义一个名为m的方法： 
                    
                    //定义方法
                    o.m = f;
                    //调用方法
                    o.m()或o.m(x,x+2);

    方法有一个很重要的属性，在方法主体内部，关键字this的值变成了调用该方法的对象。

  •             
                function compute_area()
                {
                    return this.width * this.height;
                }
