6.6 while 语句【属于基本语句】

  • while（true）会创建无限循环
  •

                    }

6.7 do/while 语句【循环至少执行一次】

  • do循环和while不同，两点：一，必须使用do来标识循环开头，二，do循环使用分号结尾
  •                 
                    function printArr(a){
                        if(a.length ==0)
                            document.write('Empty Arr!!!');
                        else{
                            var i = 0;
                            do {
                                document.write(a[i]+'br');
                            } while(++i < a.length);
                        }
                    }
                
               javascript1.2存在bug，应避免do/while中使用continue；

6.8 for语句【核心】

  • 初始化、检测、更新是一个循环的三种关键操作
  •                 
                    for(initialize; test; increment) {statement};

                    for(var count = 0; count<10; count++){
                        console.log(count+'次数')
                    }

                    for(i = 0, j = 10; i < 10; i++,j--){
                        sum += i*j;
                    }

6.16 throw语句

  • 语句发生错误时，使用throw抛出异常，使用try、catch、finally捕捉异常，

                    if(a<0) throw new Error('a is not Nim!');
                    for(var f = 1; a>1;f*=a,a--);
                    return f;
                }
                x(1)  //1;
                x(-1) //Uncaught Error: a is not Nim!
